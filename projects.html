Released:

• Celestialib

Celestialib is a small C++ header file I made which incorporates
a lot of the equations I learnt while studying for my Astronomy
GCSE into C++ code


• Basic ASCII Engine

This is a basic graphics engine I made to create graphs.
So far I've implemented a function to draw Circles and one
to make parabolic shapes


• Hardcore24

Hardcore24 is a PaperMC plugin designed to make an easy to install,
multiverse compatible Hardcore plugin with tons of config options!


• Pumpkaboo!

Pumpkaboo! is a Halloween-themed game made in the Unity game engine,
with support for both touch controls (through the APK) and keyboard controls (through the WebGL interface) !
